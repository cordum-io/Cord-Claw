# Community

CordClaw community channels are designed to help users get support quickly and
share governance patterns safely.

## GitHub Discussions

Use Discussions for asynchronous collaboration:

- Q&A: setup help, troubleshooting, and operational questions
- Ideas: feature proposals and roadmap input
- Show and tell: real-world policy examples and demos
- Policies: governance policy design and review (manual category setup required)

Discussions URL:

- https://github.com/cordum-io/Cord-Claw/discussions

## Discord

Primary real-time channel:

- `#cordclaw` in the Cordum Discord workspace

Note: channel creation requires Discord admin access and must be completed by a
workspace admin.
